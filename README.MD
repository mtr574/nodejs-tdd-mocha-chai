# NodeJS TDD with Mocha & Chai

NodeJS Test-Driven-Development project using Mocha, Chai and more.

## Usage

1. Install Mocha globally `npm i -g mocha`
2. Other dependencies `npm i`
3. Write some tests
4. `npm test`
5. Write a code to pass the tests
6. Repeat
